# شام كاش (Sham Cash) Website Clone

A detailed clone of the [shamcashsy.com](https://www.shamcashsy.com/) website.

## Features

- **RTL (Right-to-Left) Support**: Full Arabic language support with proper RTL layout
- **Responsive Design**: Fully responsive across all devices (desktop, tablet, mobile)
- **Animated Elements**: Smooth animations and transitions throughout the page
- **Interactive Components**: 
  - Floating geometric shape with rotation animation
  - Parallax mouse-follow effect on logo
  - Hover effects on phone mockup
  - Pulsing chat button with hover states
- **Modern Styling**:
  - Gradient text effects on logo
  - Dark navy/blue background with gradient overlays
  - Teal/turquoise geometric accent
  - Professional drop shadows and blur effects

## Technologies Used

- HTML5
- CSS3 (with advanced animations and gradients)
- Vanilla JavaScript
- Google Fonts (Cairo, Tajawal)

## Structure

```
├── index.html          # Main HTML structure
├── style.css           # All styling and animations

```

## Key Design Elements

1. **Logo**: Large gradient text "شام كاش" with blur shadow effect
2. **Geometric Shape**: Animated teal triangle in top-right corner
3. **Typography**: Arabic fonts (Cairo, Tajawal) with proper weights
4. **Color Scheme**:
   - Background: Dark navy (#0f1419, #1a2332, #0d1620)
   - Accent: Teal gradient (#2d7a7a, #45b5aa, #6dd9cc)
   - Logo: Blue gradient (#7b9fc4, #a8c8e9, #c8dbf0)
   - Text: White with various opacities
5. **Phone Mockup**: 3D perspective transform effect
6. **Chat Button**: Fixed position, white rounded button with pulse animation

## Animations

- **fadeInDown**: Logo entrance animation
- **fadeInUp**: Content entrance animations
- **float**: Geometric shape floating animation
- **pulse**: Chat button pulsing effect
- **Mouse tracking**: Logo follows mouse movement
- **Hover effects**: Scale and transform on interactive elements

## Browser Compatibility

- Chrome/Edge (latest)
- Firefox (latest)
- Safari (latest)
- Mobile browsers (iOS Safari, Chrome Mobile)

## Usage

Simply open `index.html` in a web browser to view the website. No build process or dependencies required.
